# RunUp Backend 🏃

API REST backend para la aplicación Android RunUp, una app de seguimiento y planificación de entrenamientos de running.

## Tecnologías
- PHP 8.2 + Laravel
- MySQL
- Claude API (Anthropic) — planificación de entrenamientos con IA
- Google Maps SDK
- Docker

## Funcionalidades
- Autenticación de usuarios
- Registro y gestión de entrenamientos geolocalizados
- Generación de planes de entrenamiento con IA
- Menús de alimentación personalizados con IA
- Notificaciones push (FCM)
- Panel de administración
